# Bulat Calendar - Приложение для управления заказами натяжных потолков

## ✅ ЧТО ГОТОВО:

- ✅ Календарь как на iPhone (месячный вид)
- ✅ Зелёные точки = замер, красные точки = монтаж
- ✅ Форма ввода данных (ФИО, адрес, номер, смета в рублях)
- ✅ Загрузка и хранение файлов (КП, дизайн и т.д.)
- ✅ Поле для ссылки на 3D проект
- ✅ Заметки по заказу
- ✅ Firebase интеграция для облачного хранилища
- ✅ PWA (можно установить на рабочий стол как приложение)
- ✅ Автосохранение в облако

---

## 🚀 КАК РАЗВЕРНУТЬ:

### ШАГ 1: Получить Firebase ключи

1. Открыть https://console.firebase.google.com
2. Выбрать твой проект "Bulat Calendar"
3. Перейти в Project Settings (шестёренка)
4. Скопировать Web SDK code
5. Это будут твои ключи

### ШАГ 2: Добавить Firebase ключи в код

1. Открыть файл `bulat-calendar-app.jsx`
2. Найти этот блок:
```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```
3. Заменить значения на свои из Firebase
4. Сохранить файл

### ШАГ 3: Загрузить на GitHub

1. Создать новый репозиторий на GitHub (https://github.com/new)
2. Назвать его `bulat-calendar` (или любое имя)
3. Загрузить все файлы проекта в репозиторий

### ШАГ 4: Развернуть на Vercel

1. Открыть https://vercel.com
2. Нажать "Import Project"
3. Выбрать репозиторий `bulat-calendar`
4. Vercel автоматически создаст ссылку (типа: https://bulat-calendar.vercel.app)

**ГОТОВО! Приложение живет в интернете.**

## 📱 КАК ИСПОЛЬЗОВАТЬ:

### На компьютере:
- Открыть ссылку в браузере
- Нажимать на дату
- Заполнять форму (ФИО, адрес, номер, смета, статус)
- Загружать файлы
- Добавить ссылку на 3D (если есть)
- Написать заметку (если нужна)
- Нажать "Сохранить"

### На телефоне (iPhone/Android):
- Открыть ссылку в браузере
- Нажать меню браузера → "Добавить на главный экран"
- На рабочем столе появится иконка
- Нажимать на иконку как на приложение

### После сохранения:
- Все данные автоматически сохраняются в Firebase
- Данные видны на любом устройстве
- Файлы хранятся в облаке (10 GB бесплатно)

## 🔧 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### Ошибка: "Firebase не инициализирован"
- Проверить, что Firebase ключи добавлены правильно
- Убедиться, что Firestore и Storage включены в Firebase консоли

### Ошибка: "Файлы не загружаются"
- Убедиться, что Firebase Storage включен
- Проверить права доступа в Firebase (Rules)

### Ошибка: "Приложение не открывается"
- Нажать F12 в браузере → Console
- Смотреть, какие ошибки выводит
- Написать мне ошибку

## 🎨 КАСТОМИЗАЦИЯ:

### Изменить цвета:
1. Открыть `BulatCalendar.css`
2. Найти `.dot.zamery` и `.dot.montaz`
3. Изменить `background-color`

### Изменить месячный вид:
1. Открыть `bulat-calendar-app.jsx`
2. Найти `monthNames` и `dayNames`
3. Изменить на русские/английские названия

### Добавить новые поля в форму:
1. В `bulat-calendar-app.jsx` найти `formData`
2. Добавить новое поле в `useState`
3. Добавить `<input>` в форму
4. Сохранить в Firebase

## 💾 ДАННЫЕ В FIREBASE:

**Firestore (база данных):**
- Коллекция: `orders`
- Документы: заказы с полями (fio, address, phone, smeta, status, date, link3d, notes)

**Storage (хранилище файлов):**
- Папка: `orders/[ORDER_ID]/`
- Файлы: все загруженные файлы по заказам
